# Mimicra

Webカメラで顔をトラッキングし、VRMモデルにリアルタイム反映するツールです。
まばたき・口パク・頭の傾きが動きます。カメラ映像・VRMファイルは一切サーバーに送信されず、すべてブラウザ内で処理されます。

## 構成

```
packages/
  vrm-core/     ← VRM表示・操作の共通エンジン(再利用モジュール)
  face-tracker/ ← Mimicra本体(顔トラッキングアプリ)
```

## ローカルで動かす

```
pnpm install
pnpm dev
```

`http://localhost:5173` をブラウザで開いてください。

## 使い方

- **カメラ選択**: 画面上部のバー（マウスを動かすと表示されます）でカメラを切り替えられます。
- **VRM読み込み**: 「VRM読み込み」ボタン、またはウィンドウに `.vrm` ファイルをドラッグ&ドロップして読み込めます。
- **背景切替**: 単色(色も選べます)・グリーンバック(`#00FF00`)・透過の3種類から選べます。
- **UIの自動非表示**: 配信中に画面を隠さないよう、しばらくマウスを動かさないとコントロールバーは自動的に消えます。マウスを動かすと再表示されます。

## OBSで使う(配信への取り込み)

1. OBSで「ソースを追加」→「ブラウザ」を選択
2. URLに、デプロイ済みのページURL（下記参照）またはローカルの `http://localhost:5173` を入力
3. 背景を「透過」に設定すると、OBS側で背景を透過合成できます（背景色に合わせてクロマキーを使う場合は「グリーンバック」を選択してください）
4. 幅・高さはお好みで調整してください

## GitHub Pagesへのデプロイ

このリポジトリには GitHub Actions によるデプロイ設定 (`.github/workflows/deploy.yml`) が含まれています。

### 手順

1. GitHubで新しいリポジトリを作成する(**Private** 推奨)
2. このフォルダの内容をpushする

   ```
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin <あなたのリポジトリURL>
   git push -u origin main
   ```

3. GitHubのリポジトリ画面で **Settings → Pages** を開き、「Source」を **GitHub Actions** に設定する
4. `main` ブランチにpushすると自動でビルド・デプロイされます(Actionsタブで進捗を確認できます)
5. デプロイ完了後、Pages設定画面に表示されるURLでアクセスできます

### 補足

- MediaPipeのモデルファイルはCDN(`jsdelivr` / `storage.googleapis.com`)から読み込むため、GitHub Pages上でもそのまま動作します。
- 公開ページにアクセスした人は誰でもカメラ許可を求められます。カメラ映像・読み込んだVRMファイルはその人のブラウザ内で完結し、外部には送信されません。
